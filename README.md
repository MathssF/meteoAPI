# Projeto Meteo-API

Este Projeto visa pegar as API do Meteomatics (https://www.meteomatics.com/en/weather-api/), e criar uma persistência própria, e assim minha própria API.

Durante este desenvolvimento irei relatar o que foi feito aqui.


# Étapas de Desenvolvimento:

### 1- Instalação

Instalei o Nest, Prisma, Docker, Swagger, e outros recursos iniciais.

### 2- Configuração do Prisma

Gerei o Prisma Schema, e configurei o banco de dados. Também to atualizando o ReadME neste ponto.
